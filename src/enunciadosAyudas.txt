primer commit
Despues de clonar
git remote remove origin
git remote add origin http://github..
git add .
git commit -m "primer cambio"
git push origin master o en cambio
git push --set-upstream origin main
Cuando quieras hacer un pull request, puedes usar:
git fetch upstream
git rebase master
Si hay conflictos, resuelvelos y luego usa git add para agregar los archivos solucionados. Luego usa
git rebase --continue
